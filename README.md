# HDI Predictor

## Project Overview
HDI Predictor is a machine learning-based web application that predicts the Human Development Index (HDI) category of a country based on input indicators such as life expectancy, education, and income.

## Features
- Predicts HDI category using machine learning.
- Visualizes HDI-related data.
- User-friendly web interface.
- Helps analyze factors affecting HDI.

## Technologies Used
- Python
- Machine Learning (Scikit-learn)
- Pandas
- NumPy
- Matplotlib
- Flask
- HTML, CSS, JavaScript
